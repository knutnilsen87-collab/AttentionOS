Important implementation decision
- The first attempt used Next.js scaffolding, but npm/package-manager installation became unstable on this Windows workspace and left partial node_modules states.
- The final MVP intentionally uses no external runtime dependencies.
- This keeps the pilot runnable with plain Node and avoids blocking project progress on package installation.
