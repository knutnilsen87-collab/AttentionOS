# DEFER_OR_DROP

## Features to defer
- Native mobile companion.
- Full recommendation-platform integrations.
- Billing/subscription.
- Organization/team mode.
- Heavy AI provider integration.

## Features to drop
No product concepts are permanently dropped yet.

## Features to freeze
- New visual redesign work.
- Additional speculative dashboards.
- Dependency-heavy framework migration.

## Why they should not consume current effort
The highest risk is not surface area. It is trustworthy data capture, persistence, auth and privacy. Those must come first.
