# PROJECT_IDENTITY

## Project name
AttentionOS

## One-line description
A Personal Attention Operating System that helps users align digital consumption with identity, focus areas, passions and discovery level.

## What the project is supposed to be
A product that scans attention inputs, calculates an Attention Alignment Score, simulates better feed direction and generates weekly reflection guidance.

## Current perceived stage
Verified MVP scaffold / pilot prototype.

## Who the project is for
Ambitious professionals, creators, entrepreneurs, students, lifelong learners and personal growth users.

## Commercial or internal purpose
Commercial subscription-product direction, currently implemented as a local MVP scaffold.

## Original goal
Build a product that helps users regain control over recommendation systems and attention allocation.

## Current goal now
Move from verified prototype to durable MVP with persistence, auth, consented extension capture and production-grade operations.

## What changed over time
The project started from prework/spec templates. It now has a runnable dependency-free prototype after package-manager instability made a dependency-heavy Next.js scaffold unsuitable in this workspace.

## Why this pack is being used now
To preserve truthful project state, avoid confusion between planned and implemented features, and give the next developer/GPT a clean continuation path.
